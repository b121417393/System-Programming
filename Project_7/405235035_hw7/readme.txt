使用程式前需先使用指令make生成 hw7-1 及 hw7-2

輸入./hw7-1 ./hw7-2即可執行程式

在hw7-1中，主程式會呼叫4次的fork。
在第一個fork時，就只有主程式能呼叫產生child，所以在第一個fork後會產生一個子程序。

而在第二個fork時，會有兩個程序能夠呼叫產生child，分別是主程式和第一個fork產生的子程序，所以在第二個fork之後會再產生2個子程序出來。

而第三個fork會有四個程序呼叫產生child，分別是主程式和第一個fork產生的及第二個fork產生的子程序，所以在第三個fork後會再產生4個子程序。

而第四個fork同理，會再產生8個子程序出來。

所以若不包含原先的主程式的話，執行主程式後所呼叫四次的fork總共會產生1+2+4+8=15個子程序出來。這些都能夠在從執行hw7-1後的輸出結果觀察得出來。

在hw7-2中，因為execle的函數參數設定，所以在執行hw7-2後，最多能夠接受的執行檔參數為6個，這6個並不包含執行檔本身，例如輸入 ls -a -l -h，這個指令的執行參數為3個。而在code的撰寫中，我所設定的最大數量為6個，超過6個的話，之後的參數將不會被列入執行依據，這個數量也可以再作擴大，但程式碼會略嫌稍長，所以就暫時先設定為6個，此程式也接受當前目錄的執行程式為參數，例如輸入./hello就會執行當前目錄下的hello執行檔。
